webslide是什么?

  1. 网络幻灯片的生成器，其目的主要是简单化网络幻灯片的制作过程。
  2. 该应用内置多种页面切换特效可以使作品更多绚丽多彩。同时由于其高扩展性，用户可以根据自己的需求去定制特效。

webslide有什么特点？

  1. 控制简单
  2. 较高扩展性
  3. 自由定制
  4. 对搜索引擎友好

如何使用?

  1. 创建一个元素，当作整个幻灯片的容器
  2. 在这个容器下每一个直接子元素就是一张幻灯片，构造好子元素，使用css来控制子元素的显示
  3. 用js来实现幻灯片整个过程的控制
  4. 这个有简单的例子: 
    
	// new 一个实例
	var slide = new WebSlide($('#webslide'));
	// 设置页面的切换特效
	slide.pageLeaveAction = "leaveFromCenterAndRotate";
	// 设置页面中每一小动画的特效
	slide.setPageStepActions("li",{
		originStatus:{opacity:0},
		targetStatus:{opacity:1},
		time: 2000
	});
