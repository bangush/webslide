What is it?

  1. A web application
  2. Help people make theri web slides easier
  3. An web slide producer

What is the feature

  1. Easy control
  2. Mark pencil
  3. Outline show
  4. There are many effects to be used
  5. Friendly to search engine

How to use it?(please press h to get more help)

  1. Use HTML and CSS to control the structure of slide. Create a wrap element, the children of the element are the slides of this application.
  2. You can use further control width javascript.
  3. There is an example: 
    
	// initialize an instance
	var slide = new WebSlide($('#webslide'));
	// set page leave action
	slide.pageLeaveAction = "leaveFromCenterAndRotate";
	// set page Step action
	slide.setPageStepActions("li",{
		originStatus:{opacity:0},
		targetStatus:{opacity:1},
		time: 2000
	});

End
